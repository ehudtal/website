What is a Mesh?

Our network is built using WiFi routers spread across the city. These routers are the "nodes" of our mesh network. A mesh network has no central server and no single internet service provider.

Our Mission

We aim to build a resilient system that will work for daily use as well as during emergencies, power outages and internet disruptions. We will run software for the community that is useful for both everyday needs and emergencies.

Get Involved

To join the mesh you can create a new node by simply buying a small router for less than $100 with no monthly fees. We will help you configure and install the device.

We meet every Sunday in downtown Manhattan. Occasionally we meet with other related organizations. Please join our meetup group for more information. We welcome people from all backgrounds.
