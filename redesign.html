A new, free network owned and operated by our community.

What is Mesh

Our network is built using WiFi routers spread across the city. These routers are the "nodes" of our mesh network. A mesh network has no central server and no single internet service provider.

What We Do

We are aiming for a resilient system that will work for daily use and also can be accessed during emergency, power outages and internet disruption. The network is free as in "free speech". We will be running software for the community that is useful both for everyday needs and emergencies.

Get Involved

  • New Node

    To join the mesh you can create a new node by simply buying a small router for less than $100 and no monthly fees. We will help you configure and install the router.
